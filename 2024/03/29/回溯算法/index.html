组合总和三

问题描述：求和为n，个数为k的组合

例如：k=3, n=9 [ 1, 2, 6]

[2,1,6]不是组合

解法1：

​ 暴力算法：k为多少就嵌套多少层for循环，然后相加判断和是否为n

解法2：

​ 递归+回溯：每一层递归就相当于一层for循环，k=2就递归2层，k=n递归无法精确控制到n层，就需要用回溯算法控制控制

参数含义

​ k：控制树的深度

​ n：控制树的宽度

​

回溯三步曲

1.递归函数的返回值一般为void，确定参数：看最后一层是一个组合（从根结点到叶子结点的路径）就是一个一维数组，还需要一个二维数组（存放收集到的所有的一位数组）返回出去。

参数：

​ path：一维数组

​ targetSum：目标组合数组大小

​ k：组合的大小（相当于for循环的遍历次数）

​ sum:当前路径已有的和

​ startIndex：决定当前递归的结点开始遍历的位置

​ 例如：根结点取1后，子节点就要从2开始取

2.回溯函数终止条件

叶子节点：

path.size() == k

返回结果前还需要判断是否是目标和

3.单层搜索的过程（每层的每个结点都是for循环）

回溯法的搜索过程就是一个树型结构的遍历过程，在如下图中，可以看出for循环用来横向遍历，递归的过程是向下遍历，回溯就是向上回溯

剪枝操作：

​ 1.判断sum> targetSum,如果满足则直接返回

​ 2.如果当前取的元素只能取到一个元素，满足不了取到k个元素。

​ path.size():当前递归的大小

​ k-path.size():当前还需要的元素

​ for循环中的i（startIndex）控制的是遍历的起始位置

​ 9-（k-path.size()）+1:至少要从这个位置开始取数，否则就k<2

​ 如果当前k=2，则至少i要从8开始取，否则就没意义

电话号码的字母组合

题目：电话号码0-9，每个数字代表3个字母的字符串，求不同数字的组合

输入 2,3

返回：2,3上面的字符串的字母组合

数字->字符串：要做映射

​ 1.map

​ 2.数组：数组的每一个元素都是字符串类型

解法：

​ 1.暴力：遍历1中的字符串，再遍历2中的字符串，再进行组合

​ 缺点：无法控制for循环

​ 2.回溯算法暴力求解，通过嵌套的方式实现n层for循环

分析：

​ 1.树的深度取决于输入数字的个数

​ 例如：2,3->深度为2

​ 2.树的宽度取决于每个数字控制的字母的长度控制

​ 例如： 2控制[d,e,f]->宽度就为3

回溯三步曲：

1.参数

​ s:取当前递归时取的字母（例如遍历数字1对应的字母取a）

​ result: 一维数组，存放叶子结点的收集结果

​ digits：String类型，输入的数字的字符串

​ index：遍历到哪个传入的数字（本层递归遍历的下标）

2.终止条件

index == digits.size()

当index 指向null时才是真正的结束

到达叶子结点，收获结果

3.单层遍历逻辑

​ 1.先把输入的数字的字符串转换成数字：int digit = digits[index] - ‘0’;

​ 2.从数组中找到对应的字母组合

​ 3.取每一个结点的所有元素（用for循环），并且i从0开始，获取每个结点，用s来存储当前获得的字母。然后再往下递归（index+1：下一层递归要指向下一个字母集合代表的数字开始递归）。然后回溯（ad->ae）

2.得到的集合要去重，比如[1,7] [7,1]

不允许重复使用nums的元素

树层去重

树枝去重

根据上图所示

若进行树枝去重，选取[1,1]不需要进行去重，因为集合里面就是有2个1

若进行树层去重，当target为3时，发现有两个分支会选中[1,2],则需要进行去重

所以遇到包含重复元素的集合时，需要先进行排序，把重复的元素放在相邻位置，当其中一个分支取了重复的元素的第一个时，那么第二个分支就不需要再去选之后的下一个相同元素（因为第一个分支选取后，进行下面的分支时，会把重复元素的情况也包含进行）

综上所述：去重的关键在于树层去重

位置：在for循环中

代码：

if(i>0 && nums[i]=nums[i-1]&&used[i-1] ==0){

​ 去重逻辑

}

其中

nums[i]=nums[i-1]：表示当前分支取的值与前一个分支取的值进行比较

used[i-1] ==0：如果used[i-1] == 1 则说明当前比较的是一个分支上的元素进行比较

ums[i]=nums[i-1]&&used[i-1] ==0：判断当前分支与前一个分支进行比较是否相等，并且判断是否使用过，如果没使用过则说明是一个分支上的，如果使用过则说明是不同分支上的。

回溯三步曲

1.参数

path一位数组，result二维数组,nums数组，targetSum,startIndex,used数组

2.终止条件

sum>targetSum

sum == targetSum

3.单层逻辑

for循环每一个分支

去重逻辑

元素放入path中，sum计数，used记录元素使用情况

递归（传入i+1，为了防止重复）

回溯（弹出元素，sum减，used使用情况为false）
